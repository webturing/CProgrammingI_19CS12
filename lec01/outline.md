## Add1 //Without Input

```C++
	#include <bits/stdc++.h>
	using namespace std;
	int main() {
  		int a=123;
  		int b=456;
  		cout<<a+b<<endl;

	}
```

## Add2 //With Input

```C++
	#include <bits/stdc++.h>
	using namespace std;
	int main() {
  		int a,b;
  		cin>>a>>b;
  		cout<<a+b<<endl;
  		return 0;
	}
```
